<h2 align="center">React Table Designer</h2>

## Introduction

A react table designer

- React
- TypeScript
- Vite
- Tailwind CSS
- react-dnd
- react-moveable
- zustand

## Install:

```sh
$ cd react-table-designer
$ pnpm install

# npm install
# yarn install
```

## Usage

Let's run!

```sh
$ pnpm run dev

# npm run dev
# yarn run dev
```
